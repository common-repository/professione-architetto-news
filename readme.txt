== Description ==

professione Architetto è il magazine web per chi è architetto e per chi lo vuole fare da grande.

Se pensi che le nostre iniziative siano interessanti e hai voglia di sostenere il nostro impegno quotidiano, puoi condividere sul tuo sito le notizie pubblicate su professione Architetto, aggiornate in tempo reale tramite feed RSS.

Grazie a questo widget si possono mostrare in tempo reale tutte le ultime notizie pubblicate o anche solo quelle di alcune categorie a scelta (Eventi, Formazione, Progetti di Architettura, Viaggi, Concorsi di Architettura, Professione).
Si può impostare il numero massimo di notizie da mostrare, la presenza del solo titolo o di un'immagine, del dettaglio della notizia o della data di pubblicazione.

Per integrarlo al meglio con qualsiasi sito web, l'aspetto delle singole parti è totalmente personalizzabile tramite CSS.

== Installation ==

Steps to install this plugin.

1. In the downloaded zip file, there is a folder with name 'professione-architetto-news'
1. Upload the 'professione-architetto-news' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Read the usage instructions in the description.
